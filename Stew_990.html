STEW-MAP LIVE Prototype

Welcome to the STEW-MAP Live Prototype. This page is designed to demonstrate, the interface designed to improve the data collection process of STEW-MAP. This tool will allow us to continuously collect data and utilise user input to control data quality. We used the IRS List of organizations eligible to receive tax-deductible charitable contributions data set to populate the map below. Currently this data is missing some enteries.
